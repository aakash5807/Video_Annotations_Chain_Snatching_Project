This project uses Hard Negative Mining for chain snatching detection.
This repository contains the implementation of a Hard Negative Mining–based CNN classification model developed during our internship for the Chain Snatching Detection project. The objective of this model is to improve detection robustness by focusing training on difficult negative samples that closely resemble positive chain snatching cases. Instead of treating all negative samples equally, the system identifies misclassified or high-loss negative samples and gives them higher importance during retraining using a weighted sampling strategy. This approach reduces false positives, improves decision boundary learning, handles dataset imbalance effectively, and enhances real-world performance in CCTV-based crime detection scenarios.
